# Doctor Appointment

Simple PHP/JavaScript appointment booking project intended to run on XAMPP (Apache + PHP + MySQL).

## Quick start

- Place the project in your XAMPP `htdocs` folder (already set).
- Start Apache and MySQL from the XAMPP control panel.
- Open `http://localhost/Doctor-Appoinment/` in your browser.

## Structure

- `index.php`, `index.html` — entry pages
- `css/` — styles
- `js/` — client-side JS (appointments, booking, dashboard, etc.)

## Notes
- Update database credentials in your PHP config before running.
- Add more documentation as you develop features.
